## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is configured for static hosting on Netlify. The built files are in the `dist/` directory after running `npm run build`.
